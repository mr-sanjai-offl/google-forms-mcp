# Support Guide for Google-Forms-MCP

Thank you for using **Google-Forms-MCP**! We want to ensure you have the best experience automating your Google Workspace workflows with AI assistants.

## Getting Help

### 1. Documentation & Guides
Before submitting an issue, please review our comprehensive guides:
- **[README.md](README.md)**: Installation, quickstart, and configuration guide.
- **[Examples Directory](examples/README.md)**: 10 starter workflow templates and prompt engineering guides.
- **[Troubleshooting & FAQ](README.md#troubleshooting)**: Solutions for OAuth errors, rate limits, and client connectivity.

### 2. Community Discussions & Q&A
Have a question about prompt workflows, custom templates, or integrating with specific MCP clients (Claude Desktop, Cursor, Cline, Roo Code)?
- Visit our **GitHub Discussions** page (or open a General Q&A issue if GitHub Discussions is disabled on your fork).
- Search previous discussions to see if your question has already been answered.

### 3. Reporting Bugs
If you encounter a reproducible bug, please check existing open issues first. If your issue is new, open a bug report using our **[Bug Report Template](.github/ISSUE_TEMPLATE/bug_report.yml)**. Be sure to include:
- Your operating system and Python version (`python --version`).
- The MCP Client being used (e.g., Claude Desktop v0.7.0, Cursor v0.42).
- Relevant log output from stderr (ensure all sensitive tokens are redacted!).

### 4. Security Vulnerabilities
**Do not open public issues for security or credential leaks.** Please follow our strict disclosure procedure outlined in **[SECURITY.md](SECURITY.md)**.

## Commercial & Enterprise Support
For organizations requiring custom integrations, SLA-backed support, or private deployment consulting for Google Workspace AI agents, please reach out via GitHub Sponsors or community maintainer contact channels.
